dentro del que es dependencias se agregaron:

1. date-fns: se agregó para poder dar formato a la fecha y hora de los datos de compra enviados a FIREBASE.

4. sass: se utiliza la dependencia para trabajar con SASS directamente en la herramienta.

5. firebase: se utiliza la conexión directa a la base de datos de firebase.
